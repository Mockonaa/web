Comment rejoindre le serveur avec un launcher crack.

1. Récupérer le launcher

récupérez le launcher crack ici.

2. Installer le Jeu (sans le modpack)

Démarrez le launcher récupéré précédemment:

-> Soit en double cliquant dessus

-> Soit en ouvrant un terminal à l'emplacement du launcher, et en exécutant la commande suivante: 

java -jar launcher.jar

Ensuite entrez le pseudo de votre choix dans la case "User name"

Pour la version sélectionez "Release 1.12.2"

Après vous pourrez cliquer sur "Install and play"

Maintenant patientez que le jeu se télécharge et se lance.

Quand le jeu sera lancé, veuillez fermer le jeu pour pouvoir installer le modpack ^^

3. Installer le modpack sur votre Jeu

Premièrement, téléchargez le modpack en cliquant ici

Une fois téléchargé, allez dans votre launcher, sélectionnez l'icone de dossier située en bas à droite, cela aura pour effet d'ouvrir votre gestionnaire de fichier, à l'emplacement des fichiers du jeu. Décrompréssez modpack.zip dans le dossier ouvert.

Après il ne reste qu'une seule chose! lancer le jeu, mais avant de cliquer sur "play", changez la version à "Forge 1.12.2" Sinon le modpack ne chargera pas :/

Enfin cliquez sur jouer, et c'est terminé!
